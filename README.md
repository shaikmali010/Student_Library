# Student Library System

A Library Management System built using:

- Java
- JDBC
- MySQL
- Eclipse

Features:
- Admin Login
- Add/Search/Update/Delete Students
- Add/Search/Update/Delete Books
- Issue and Return Books
- JDBC Database Connectivity

Project Structure:

src/
 ├── model
 ├── service
 ├── util
 ├── Main.java
 └── DBConnection.java
